Hi and welcome to wordweaver. This is a mini project using HTML, CSS, and p5.js to create an elmentary cellular automata. It will convert your text input to color by partitioning the character data. After the first "strip", it'll apply an iterative rule to your pattern, forming a higher-order behavior. What is that iterative rule? how does it work? uhhhh idk lol


Please note that 1) this uses UTF-16 codes, and 2) all script run clientside. if you are not a computer nerd you do not have to worry about what this means but if you are a nerd you have been warned. Will use only the first 15 chracters!






Settings:

Since the actual colors displayed are arbitrary, there's no reason why you can't customize them to your liking! Each value is a multiplier to the character code. Don't be afraid to experiment with different spacings and value sizes!
